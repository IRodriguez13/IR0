# IR0 0.0.1 — Release scope (honest inventory)

> **Last verified:** 2026-06-26  
> **Source of truth:** `make smoke-release-0.0.1`, `Documentation/releases/IR0_0.0.1_SCOPE.md`,  
> `scripts/linux_abi/contracts.json`, `kernel/test/`, `tests/host/`, Makefile smoke targets

## Release gate (D1.20)

```bash
make smoke-release-0.0.1    # primary 0.0.1 gate
make release-0.0.1            # + kernel-text-budget
```

**Included gates (deterministic):**

| Gate | Validates |
|------|-----------|
| `roadmap-phase1-stability` | `kernel-x64.bin`, host tests, `kernel-tests`, memsafe, analyze, matrix-min, arch-guard, `smoke-mm-cow-lazy` |
| `linux-abi-audit` | Linux↔IR0 `brk` ground truth + host/ktest evidence |
| `smoke-runit-ash-interactive` | runit + BusyBox ash + TTY canonical read (`echo hi`) |
| `smoke-fat16-mount` | FAT16 read-only on `/dev/hdb` |

**Explicitly removed from release gate (legacy / flaky / diagnostic):**

- `smoke-fase52-tcc` — **WARN/BLOCKER** if required manually; hangs at first static link in QEMU (see STABLE.md)
- `smoke-fase55*` Doom stubs — optional tier-2; not part of 0.0.1 core
- `IR0_LEGACY_SMOKE=1` fase53b-posix-pseudofs — superseded by ktests + tier smokes

## 1. Features included in 0.0.1

### MM / mmap / brk

| Item | Status | Gate | Limitations |
|------|--------|------|-------------|
| Lazy anon mmap (`CONFIG_LAZY_ANON_MMAP`) | **VERIFIED** | `smoke-mm-cow-lazy`, ktest `mmap_null_placement` | No MAP_SHARED file-backed COW at Linux parity |
| Lazy brk heap (`CONFIG_LAZY_BRK_HEAP`) | **VERIFIED** | same + ktest `brk_post_exec`, `linux-abi-audit` | Initial break layout differs from Linux absolute VA; delta semantics match |
| mmap NULL top-down + stack gap | **VERIFIED** | host + ktest `mmap_null_placement` | Gap size policy in `mmap_contract.h`; not full Linux ASLR |
| Fork COW (hybrid) | **VERIFIED** | `smoke-mm-cow-lazy` FASE40 A–F | No 2 MiB huge-page COW |
| PMM frame refcount | **VERIFIED** | ktests + FASE40 smoke | — |

| Item | Status | Gate | Limitations |

## 4. Known risks

| Risk | Severity | Mitigation |
|------|----------|------------|
| **ash interactive smoke flaky** | **BLOCKER** if fails on release run | Fail release; do not hide. Target 3/3 local runs before tag |
| **fase52-tcc hang** | **WARN** (out of gate) | Manual `build-tcc-fase52`; not claimed stable |
| **FAT16 read-only** | INFO | No write path in 0.0.1 |
| **linux-abi-audit: brk only** | INFO | Enable next contracts incrementally |
| **KTM_EVENTS serial noise** | LOW | Acceptable; not D1 forensics |
| **FASE40_D_AUDIT dead blocks in `process.c`** | LOW | Remove in follow-up ARCH-4 sprint |
| **FAT16 serial tag split** | LOW | Short `[FAT16OK]` tag + flat grep in Makefile |
| **Uncommitted D1.18/D1.19/D1.20 worktree** | PROCESS | Commit before release tag |
