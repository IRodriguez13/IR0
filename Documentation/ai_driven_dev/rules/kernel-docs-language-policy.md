# IR0 Documentation Language Policy

## Default Language

- Keep technical documentation and code comments in English by default.

## Spanish Documentation

- Spanish versions are allowed only in a dedicated `/esp` directory.
- For kernel docs, place translations under `Documentation/esp/`.
- Do not mix English and Spanish in the same primary technical document.

## Consistency

- Ensure translated docs stay aligned with implemented behavior.
- Avoid aspirational wording for features that are not implemented.
