# IR0 0.0.1 — VFS write-path bundle (R3-vfs-write-path-bundle)

> **Last verified:** 2026-06-26 — **VERIFIED** (`make linux-abi-audit-vfs-write`)  
> **Source of truth:** `scripts/linux_abi/workloads/vfs_write_probe.c`,  
> `make linux-abi-audit-vfs-write`, `scripts/linux_abi/contracts.json`

## Meta

Auditar el **write-path VFS completo** en una sola oleada (contract bundle).  
Backend primario validado: **MINIX rw** en `/tmp/ir0wtest` (tmpfs fallback si `/tmp` no es escribible).  
**FAT16 rw:** re-ejecutar el mismo bundle **sobre disco FAT**; no implementar write FAT16 hasta PASS.

### Primer gap FAT16 (2026-07-10, code source of truth)

| Hecho | Evidencia |
|-------|-----------|
| Mutators stub | `fat16_disk_write` / `create` / `mkdir` / `unlink` / `truncate` → `fat16_rofs()` → `-EROFS` (`fs/fat16_disk.c`) |
| Audit actual | `run_ir0_vfs_write.sh` formatea **MINIX** efímero; no monta FAT |
| Gate RO | `smoke-fat16-mount` PASS (mount + `HELLO.TXT` read) |
| Desbloqueo | Nuevo runner (o flag) que monte FAT16 rw-capable + `linux-abi-audit-vfs-write` PASS; luego impl mínima |

## Scope (29 pasos)

openat (O_CREAT/O_EXCL/O_TRUNC/O_APPEND) · write/read roundtrip · lseek · unlink · rename · mkdir/rmdir · truncate/ftruncate.

## Fixes cerrados (2026-06-26)

| Gap | Fix |
|-----|-----|
| unlink_dir errno | `vfs_unlink` → `-EISDIR` (Linux parity) |
| rename en tmpfs | `tmpfs_link` para `vfs_rename` |
| IR0 runner disk | `run_ir0_vfs_write.sh`: MINIX efímero `--format-large` (no `cp disk.img`) |
| MINIX truncate grow | `minix_inode_zero_range` + persist `i_size` |
| ftruncate | `sys_ftruncate` → `vfs_truncate(path)` vía fd_table |
